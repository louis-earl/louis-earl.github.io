Best practices using classes & ids

In CSS, the class and id selectors can be used to style specific HTML elements. In this post, we’ll take a quick look at how these selectors are used, and describe the best practices for when to use classes instead of id’s, and vice-versa.

Class Selector

The class selector is used for styling multiple HTML elements.
There are a couple of ways to use the class selector. Firstly, the broad approach - used by a period symbol, followed with the class name.
For example:

CSS:

                    
    .emphasis-text {
    font-weight: bold;
    }

HTML:

                    
    <h1 class=“emphasis-text”>This is a purple title</h1>

    <p>
        Wow, that is a <span class=“emphasis-text”>purple</span> word!
    </p>

Result:

This is a purple title

Wow, that is a purple word!
