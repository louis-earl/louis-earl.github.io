An analogy to describe the differences between HTML and CSS

Say you were building a house. HTML would be all things structural - the foundations, wooden floor joists, wall studs, roof rafters, windows, etc. This is the framework, the skeleton of the house.

Then we add the CSS to our house. This is the part where everything we do is cosmetic. What colour are we painting the walls? Does the living room have carpet or wooden floorboards? For example, driving to work:

  • Get in car
  • Start car
  • Begin driving
  • While not at work:
    • Move forward
    • If at intersection:

Describe what the DOM is and an example of how you might interact with it

The Document Object Model, in the context of web development, is the interface used when accessing and manipulating HTML documents. It defines the logical structure of the document, so that programmers can navigate the structure, add, modify and remove elements and content.

Developers can interact with the DOM using JavaScript, this allows them to modify and/or add functionality to HTML elements.

Explain the difference between accessing data from arrays and objects.

Objects are used to represent a’ thing’ in code. For example: A high school student object could store associated data such as:

  • Name
  • Age
  • Email address
  • Subjects enrolled in

Arrays should be used to store multiple things of the same type. For example, within our student class, the subjects enrolled in should store multiple subjects:

  • English
  • Mathematics
  • ICT
  • Physics
  • Media Studies

We’ll use the following student object for the next examples (I won’t go into the detail of how to actually make the class): 

                    
    // Pretend this is an existing student object
    student:
        name = “Louis”;
        age = 23;
        email address = louis@test.com
        // This is an array!
        subjects = [“English”, 
            “Mathematics”, 
            “ICT”, 
            “Physics”, 
            “Media Studies”]

To access their age. We use dot notation like so:

var name = student.name

So name is now equal to the string, “Louis”.

To access the list of all their subjects we would use dot notation to get the array:

var allSubjects = student.subjects;

To access the second subject from the subject array, we use bracket notation. Note that arrays start counting at zero, so the first item in the array is at index[0].

var secondSubject = student.subjects[1];

So now secondSubject is equal to the string, “Mathematics”.

Explain what functions are and why they are useful

A function is a block of code designed to perform a single action. They are useful because functions are reusable. If you have to perform the same action multiple times, you can simply call that function, rather than having to type out the exact same code multiple times.
